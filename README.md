# api-go
  api-go est une API de visualisation de BjNet Monitor
 
# Mise en place
  
  ## Installer Golang
    https://golang.org/doc/install
  
  ## Clone repository
  ```bash
  https://github.com/Abousidikou/api-go.git && cd api-go
  ```
  
  ## Initialize go module
  ```bash
  go mod init exampleModule
  ```
  
  ## Download missing package
  ```bash
  go mod tidy
  ```
  
  ## Run api
  ```bash
  go run api &
  ```
